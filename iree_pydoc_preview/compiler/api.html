In-Process Compiler API

IREE provides access to its MLIR-based compiler via a dedicated set of APIs presented here.

iree.compiler.api.driver.build_iree_vm_pass_pipeline(options: iree.compiler._mlir_libs._ireeCompilerDriver.CompilerOptions, pass_manager: MlirPassManager) → None

Populates VM compilation pass on a PassManager.

This is the primary interface to IREE’s backend compiler, providing compilation from a supported set of input dialects to the vm dialect, representing IREE’s lowest level representation.

iree.compiler.api.driver.translate_module_to_vm_bytecode(options: iree.compiler._mlir_libs._ireeCompilerDriver.CompilerOptions, module: MlirModule, file: object) → None

Given a vm.module translate it to VM bytecode.

The provided file argument must be a valid IO object, capable of having binary data written to it.
